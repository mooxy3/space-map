project naam:
space map


beschrijving:
dit is een map website waar space ship capitein snel naar andere planeten of sterren stelsel kan navigeren.


doelgroep:
schepen kapitein

in scope
een voorpagina, login, map pagina en een volledige zonnestelsel.

out of scope
het website UI mooier maken, de planeten een texture geven, een licht effect geven aan de zon.
